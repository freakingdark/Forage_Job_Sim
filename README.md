# 💼 Forage Job Simulation Projects

This repository contains practical implementations completed as part of multiple **Forage virtual job simulations**. These simulations are designed to replicate real-world business scenarios across leading organizations and provide hands-on experience in data analysis, problem-solving, and technical execution.

The projects demonstrate applied skills in Python, data analysis, and structured business reporting within simulated corporate environments.

## 🏦 JP Morgan Simulation

**Focus Area:** Financial risk analysis & data-driven decision-making  

This simulation involved working through structured tasks aligned with real-world financial workflows. The project emphasizes:

- Data cleaning and preprocessing  
- Analytical reasoning applied to financial datasets  
- Structured problem-solving  
- Deliverable presentation of findings  

Tools used:
- Python
- Jupyter Notebook
- Pandas
- NumPy
- Matplotlib
